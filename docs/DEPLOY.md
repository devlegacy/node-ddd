## Secrets

### Show

```sh
  vercel secrets list
```

### Create

```sh
  vercel secrets add [secret-name] [secret-value]
  vercel secrets add ggtech-db-user username
```

## Deploy

### Local

```sh
  vercel dev
```

### Prod

  vercel
